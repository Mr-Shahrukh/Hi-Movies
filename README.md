# 🎬 Hi_Movies

🚀 Built as part of my Fullstack Java Developer journey

A modern movie browsing web application built using React.js Users can explore movies, search for their favorite films, and manage a personalized favorites list.

---

## 🚀 Features

- 🔍 Search movies
- 🎥 Browse trending/popular movies
- ❤️ Add/Remove favorites
- 💾 Favorites saved using Local Storage
- ⚡ Fast and responsive UI
- 📱 Mobile-friendly design

---

## 🛠️ Tech Stack

- Frontend: React.js
- Styling: CSS
- State Management: Context API
- API: TMDB API
- Storage: LocalStorage

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/Mr-Shahrukh/hi-movies.git
```

2. Navigate to project folder

```bash
cd hi-movies
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```
